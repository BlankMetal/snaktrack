# Design Notes

The dashboard should feel like a dependable internal tool, not a marketing site.

The 2021 rebrand used a polished card-based visual language. The current app did not adopt that system. New UI should usually match the existing dashboard shell unless the product direction changes.

Office Operations copy should stay useful before it gets warm. Start with the actual request status, approval condition, or next operational check. The care team may ask for a softer employee-value sentence after the useful part, especially when free snacks are involved.
